# GST Calculator

A modern and responsive GST Calculator built using React and Tailwind CSS.

This project was created as a trial task for Digital Heroes and helps users easily calculate GST amounts by adding or removing GST from the original price.

##  Live Demo

https://gst-calculator-teal.vercel.app/

## GitHub Repository

https://github.com/officialsonia1018-lab/gst-calculator

##  Features

* Add GST to the original amount
* Remove GST from the total amount
* Supports GST rates:

  * 5%
  * 12%
  * 18%
  * 28%
* Intra-state GST calculation (CGST + SGST)
* Clean and responsive user interface
* Fast and easy to use

##  Tech Stack

* React.js
* Tailwind CSS
* JavaScript
* Vite
